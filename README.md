`./update-vmware-hostmodules/update-vmware-hostmodules` - скрипт, который выполняется когда vmware не хочет загружаться потому что обновилось ядро

`./hooks` - хуки которые выполняются после `pacman -S` 
    `nvidia.hook` - подстраховка, что драйвера подтянутся при включении
    `nvidia-toolkit.hook` - "обновляет nvidia" в докере"

`./monitors` - простой парсер, который из yml файла пробрасывает конфигурацию в xrand. Основное предназначение было просто хранение нескольких профилей, чтобы можно было переключаться при необходимости
